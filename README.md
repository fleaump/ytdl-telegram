# Telegram-бот для скачивания видео с YouTube

Telegram-бот, который скачивает видео с YouTube и отправляет обратно с обновлением прогресса в реальном времени.

## Возможности

- 📥 Скачивает видео с YouTube в формате MP4
- 📊 Обновление прогресса в реальном времени (скорость загрузки, оставшееся время, процент)
- 📤 Отправляет видеофайлы обратно в чат Telegram
- 🐳 Готов к развёртыванию в Docker
- 🔒 Белый список через SQLite базу данных

## Требования

- Docker и Docker Compose
- Токен Telegram-бота (получите у [@BotFather](https://t.me/botfather))

## Установка

### 1. Получите токен Telegram-бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/botfather)
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен бота

### 2. Настройте окружение

Создайте файл `.env` в корне проекта:

```bash
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_chat_id
ACCESS_PASSWORD=ваш_секретный_пароль
```

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |
| `ADMIN_CHAT_ID` | Ваш chat_id — полный доступ к управлению |
| `ACCESS_PASSWORD` | Пароль для команды `/me` (необязательно) |

> **Как узнать свой chat_id:** отправьте сообщение боту [@userinfobot](https://t.me/userinfobot)

### 3. Запустите через Docker Compose

```bash
docker-compose up -d
```

### 4. Запустите бота

Откройте Telegram, найдите вашего бота и отправьте `/start` или вставьте ссылку на YouTube.

## Использование

1. Скопируйте ссылку на видео с YouTube
2. Вставьте её в чат Telegram
3. Дождитесь скачивания (будут отображаться обновления прогресса)
4. Получите видеофайл!

## Команды бота

### Основные команды

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота |
| `/help` | Справка по использованию |
| `/me <пароль>` | Добавить текущий чат в белый список (нужен пароль) |

### Управление белым списком (только администратор)

| Команда | Описание |
|---------|----------|
| `/add <chat_id>` | Добавить chat_id в белый список |
| `/remove <chat_id>` | Удалить chat_id из белого списка |
| `/list` | Показать все chat_id в белом списке |

**Примеры:**
```
/add 123456789
/remove 123456789
/me мой_секретный_пароль
/list
```

## Ограничения

- **Максимальный размер файла**: 50МБ (ограничение API ботов Telegram)
- **Формат**: только видео MP4
- Бот уведомит вас, если видео превысит лимит размера

## Архитектура

- **Python 3.11** с `python-telegram-bot` и `yt-dlp`
- **FFmpeg** для обработки видео
- **Docker** для контейнеризации
- Отслеживание прогресса в реальном времени через редактирование сообщений

### Модульность

- **`db.py`** — управление SQLite базой белого списка. Можно заменить на другую СУБД не трогая бота.
- **`youtube.py`** — загрузчик YouTube видео. Содержит базовый класс `VideoDownloader` для добавления других сервисов.
- **`bot.py`** — только логика Telegram бота, делегирует загрузку и БД модулям.

download_manager.register(VimeoDownloader())

## Белый список

Бот использует SQLite базу данных для хранения белого списка разрешённых чатов.
База данных хранится в `/tmp/downloads/whitelist.db` внутри контейнера.

При каждом запросе бот проверяет наличие `chat_id` в базе данных.
Если чат не в белом списке — бот ответит сообщением об отказе в доступе.

### Управление доступом

- **Администратор** (`ADMIN_CHAT_ID`) — имеет полный доступ ко всем командам без ограничений
- **Обычные пользователи** — могут добавить себя через `/me <пароль>`, если задан `ACCESS_PASSWORD`
- Если `ACCESS_PASSWORD` не задан — саморегистрация отключена, только администратор может добавлять
